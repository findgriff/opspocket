The Tunnel

Encrypted SSH Tunnel.
Zero Ports Exposed.

OpsPocket punches a secure direct-tcpip tunnel over your SSH connection and opens your VPS web UIs in-app — no firewall rules, no VPN, no risk.

  • OpenClaw UI tunnel (port 18789) with auto token auth
  • Mission Control tunnel (Nginx port 80)
  • All traffic encrypted end-to-end via SSH
  • Tunnel tears down cleanly when you close the screen

Frequently Asked Questions

OpsPocket is an iPhone app for DevOps engineers and developers who need remote access to their Linux VPS. It's particularly suited to teams running OpenClaw — Anthropic's open-source AI management framework — but works great as a general-purpose SSH terminal and server management console too.
No. OpsPocket works with any Linux VPS over SSH. OpenClaw support is a bonus — all the slash commands, gateway management and Mission Control tunnel are available if you run OpenClaw, but the SSH terminal, PM2, systemd, Docker and general commands work on any server.
Extremely secure. Credentials are stored exclusively in iOS Keychain using flutter_secure_storage — the same encrypted store used by banking apps. They are never sent to any cloud server, never logged, and are cleared from memory immediately after use.
OpsPocket creates a local TCP port forward over your SSH connection (SSH direct-tcpip). Your iPhone's browser talks to a local port that tunnels traffic to port 80 on your VPS — no external ports need to be open, and all data travels encrypted through your SSH connection.
Monthly plans can be cancelled any time with no penalty. Annual and 2-year plans come with a 7-day money-back guarantee. Once committed, they run for their full term — so you lock in the savings. Contact support if you need to make any changes.
OpsPocket is currently iOS-first, built specifically for iPhone. An Android version is on the roadmap but no release date has been confirmed yet. Annual and 2-year subscribers will get early access when it arrives.
